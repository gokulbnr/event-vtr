# Event-Based Visual Teach-and-Repeat via Fast Fourier-Domain Cross-Correlation

[![Pixi](https://img.shields.io/badge/pixi-supported-brightgreen?style=for-the-badge&logo=pixi)](https://pixi.sh)
[![RoboStack](https://img.shields.io/badge/robostack-supported-brightgreen?style=for-the-badge&logo=ros)](https://robostack.github.io)

Welcome to the official repository for the paper [**Event-Based Visual Teach-and-Repeat via Fast Fourier-Domain Cross-Correlation**](https://arxiv.org/abs/2509.17287), to be presented at the 2026 IEEE/RSJ International Conference on Intelligent Robots and Systems (IROS 2024). This work teach-and-repeat demonstrates teach-and-repeat navigation on a mobile robot using an event-based camera as the only form of exteroception. Recordings from reported demonstrations are available at [QUT-Event-VTR-Dataset](https://huggingface.co/datasets/gokulbnr/QUT-Event-VTR-Dataset).
